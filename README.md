# OMSARATHY RAMESH - Filmmaker Portfolio

A modern, film-themed portfolio website with light/dark mode toggle and timeline-based experience section.

## Features

- 🎬 **Film-Themed Design**: Unique cinematic aesthetic with film strip elements, camera visuals, and film reel animations
- 🌓 **Light/Dark Mode**: Toggle between light and dark themes with persistent preference storage
- 📅 **Timeline Experience**: Beautiful vertical timeline showcasing work experience
- 📱 **Responsive Design**: Fully responsive layout that works on all devices
- ✨ **Smooth Animations**: Fade-in animations and interactive elements
- 🎯 **Modern UI/UX**: Clean, professional design with film industry aesthetics

## Sections

1. **Hero Section**: Eye-catching introduction with animated camera visual
2. **About**: Personal introduction and background
3. **Education**: Academic qualifications
4. **Skills**: Technical, Operational, and Interpersonal skills
5. **Experience**: Timeline-based work experience
6. **Projects**: Showcase of notable projects
7. **Contact**: Contact form and social media links

## How to Use

1. Simply open `index.html` in a web browser
2. No build process or dependencies required
3. All files are self-contained

## Customization

- **Colors**: Edit CSS variables in `styles.css` under `:root` and `[data-theme="dark"]`
- **Content**: Update text content directly in `index.html`
- **Theme**: Dark/light mode toggle is in the top navigation bar

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
